Getting Started
Dashboard

- Upon logging in, you are brought to the dashboard, which serves as the hub for HomeKeep. From here, you can navigate to other sections of the app.

- The dashboard by default offers a table view of all your maintenance tasks.

- Easily add new tasks using the add task (+) button.

- Each task includes details such as name, category, start date, interval, and description.

- You can edit and delete tasks to manage them effectively.

Calendar

- HomeKeep provides a calendar view in addition to the dashboard's table.

- Add tasks directly to the calendar by clicking on a date block or using the add task (+) button.

- Data between the table and calendar views is synchronized for consistency.

- Navigate through upcoming months to see scheduled maintenance tasks.

- You can edit and delete tasks directly from the calendar view.

Notifications

- Receive cloud-based email notifications when your tasks are due.

- Notifications are active as long as the task is in the table or calendar.

- Delivery is based on the interval you set for each task.

- To stop notifications for a task, simply delete it from the table or calendar.

Getting Help

- Use the help request feature for any questions or issues.

- Chat windows are available throughout the app for sending help requests.

- We aim to respond to help requests promptly to ensure a smooth experience.
