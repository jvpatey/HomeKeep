Getting Started
1. Welcome

- You will be greeted with a welcome page upon visiting HomeKeep for the first time.

- The welcome page includes a sign in form, description of the app, and a link to create an account.

- The first step is for you to create an account to personalize your experience and securely store your data.

2. Signing in

- HomeKeep supports two sign-in methods: username/password and Google sign-in.

- You can create an account via the provided form, or use your existing Google credentials.

- Accounts are essential for a personalized experience and linking maintenance tasks to each user.

3. Dashboard

- After signing in, you will be directed to the dashboard.

- The dashboard provides a table view of all maintenance tasks linked to your account.

- You can easily add new maintenance tasks to the database and table via the add task (+) button

- Each task includes details such as name, category, start date, end date, interval, and description.

- Edit and delete functionality are available for you to manage your tasks effectively.

4. Calendar

- HomeKeep offers a calendar view as an extension to the dashboard's table.

- Tasks can be added directly to the calendar by clicking on the date block, or using the add task (+) button.

- Data between the table and calendar views are synchronized to ensure consistency.

- You can navigate through following months to see scheduled maintenance based on the intervals you provide.

5. Notifications

- You have the option to turn your notifications on/off for your maintenance tasks.

- Toggle switches are available in the table view, allowing you to customize your notification preferences.

- Notifications are sent via the cloud on the day the task is due, providing you with timely reminders.

6. Getting Help

- HomeKeep includes a help request feature to assist you with any questions or issues you may experience.

- Chat windows are available to help you send help request messages.

- Responses to help requests are provided as soon as possible to ensure a seamless experience.
