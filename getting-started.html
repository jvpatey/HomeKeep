Getting Started
1. Welcome

- Users are greeted with a welcome page upon visiting HomeKeep for the first time.

- The welcome page introduces HomeKeep as an app for effortless home maintenance.

- The first step for users is to create a user account.

2. Signing in

- HomeKeep supports two signing in methods: username/password and Google sign-in.

- User accounts are essential for a personalized experience and linking maintenance tasks to each user.

3. Dashboard

- After authentication, users are directed to the dashboard.

- The dashboard provides a table view of all maintenance tasks linked to the user's account.

- Users can easily add new maintenance tasks to the database and table.

- Each task includes details such as name, category, start date, interval, and description.

- Edit and delete functionality are available for users to manage their tasks effectively.

4. Calendar

- HomeKeep offers a calendar view in addition to the table dashboard.

- Users can view their scheduled maintenance tasks on a calendar.

- Tasks can be added directly to the calendar by clicking on the date or using the "add task" button.

- Data between the table and calendar views are synchronized to ensure consistency.

- Users can navigate through following months to see scheduled maintenance based on intervals provided.

5. Notifications

- Users have the option to toggle notifications for maintenance tasks.

- Toggle switches are available in the table view, allowing users to customize their notification preferences.

- Notifications are sent via the cloud on the day the task is due, providing users with timely reminders.

6. Getting Help

- HomeKeep includes a help request feature to assist users with any questions or issues.

- Chat windows are accessible on most pages, allowing users to send help messages.

- Responses to help requests are provided as soon as possible to ensure a seamless user experience.
